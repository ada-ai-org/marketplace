# Ada AI Marketplace

Plugin marketplace for [Ada AI](https://github.com/ada-ai-org) tools.

## Install

```
/plugin marketplace add ada-ai-org/marketplace
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [ada-drive](https://github.com/ada-ai-org/ada-drive) | A shared filesystem for humans and AI agents — local-first, GitHub-synced, works with Claude Code, Codex, and any agent. |

## Install a plugin

```
/plugin install ada-drive@ada-ai
```
